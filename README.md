## 简介

微讯是一款使用 MVP + Retrofit2 + RxJava2 + Dagger2 框架开发的新闻APP。
包含新闻、视频、妹子和我的四个模块，项目大致功能已经完成，部分功能正在完善中。
如果觉得对你有帮助的话请帮忙star一下，多谢啦！！！

该项目旨在帮助开发者
