# 微讯

微讯是一款使用 MVP + Retrofit2 + Rxjava + dagger2 等框架开发的阅读软件。包括新闻、视频、妹子和我的四个模块，项目大致功能已经完成，部分功能正在完善中。如果觉得对你有帮助的话请帮忙star一下，多谢啦！
